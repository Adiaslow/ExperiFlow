# ExperiFlow

## Install NPM
`npm install -g npm`

## Install NPX
`npm install -g npx`

## Clone Repository
`git clone https://github.com/Adiaslow/ExperiFlow.git /Path/To/Directory`
## Navigate to directory
`cd /Path/To/Directory/ExperiFlow`

# Host Server
`npx http-server ./ -a 127.0.0.1 --no-dotfiles -c-1 -P http://127.0.0.1:8080`
